# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest minor | ✅ |
| Previous minor | ⚠️ (critical fixes only) |
| Older | ❌ |

## Reporting a Vulnerability

**Do NOT file a public issue for security vulnerabilities.**

Report security vulnerabilities to: **security@ubag.dev**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and aim to patch critical vulnerabilities within 7 days.

## Disclosure Policy

We follow coordinated disclosure: we will notify you when a fix is ready and coordinate a public disclosure date (typically 30-90 days after initial report).

## Scope

In scope: gateway API, worker, sidecar, SDKs, dashboard
Out of scope: third-party dependencies (report to their maintainers), infrastructure outside this repository
